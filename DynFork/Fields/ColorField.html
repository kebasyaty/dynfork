    struct DynFork::Fields::ColorField

    • DynFork::Fields::ColorField
    • DynFork::Fields::Field
    • Struct
    • Value
    • Object

    Overview

    Field for entering color.

    NOTE The default value is #000000 (black).

    NOTE Html input type="text".

    WARNING type="color" only seven-character hexadecimal notation.

    WARNING Used validator Valid.color_code?.

    NOTE Examples: #fff | #f2f2f2 | #f2f2f200 | rgb(255,0,24) | rgba(255,0,24,0.5) | rgba(#fff,0.5) | hsl(120,100%,50%) | hsla(170,23%,25%,0.2) | 0x00ffff
